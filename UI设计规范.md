# Media Manager UI 设计规范

## 设计原则

1. **用户为中心**: 所有设计决策以用户需求为出发点
2. **简洁清晰**: 去除冗余，突出核心功能
3. **一致性**: 保持视觉和交互的统一
4. **可访问性**: 确保设计对所有用户友好
5. **美观实用**: 在美学与功能间取得完美平衡

---

## 一、配色方案

### 1.1 暗色主题 (Dark Theme) - "深邃星夜"

暗色主题采用深蓝-紫色调，营造专业、沉浸的媒体管理氛围。

   - 垂直排列的分类选项

   - 封面图（16:9 或 3:4 比例）
   - 角标（媒体类型+单作/系列）
   - 名称（单行截断）
   - 作者（单行截断）
   - 标签（最多显示3个，超出+N）
   - 悬浮显示播放按钮

## 三、组件设计

### 3.1 卡片组件 (ItemCard)

#### 尺寸规范
- 宽度：自适应（响应式网格）
- 高度：固定 280px
- 封面高度：180px
- 圆角：12px

#### 状态
- **默认状态**：背景 `var(--bg-card)`，边框 `var(--border)`
- **悬浮状态**：上移 4px，阴影增强，边框变亮
- **加载状态**：封面区域显示加载动画
- **无封面状态**：显示媒体类型图标

#### 内容布局
- 角标：左上角（媒体类型）+ 右上角（系列/单作）
- 信息区：名称（1行）、作者（1行）、标签（3个+溢出数字）

## 五、动效规范

### 过渡时长

| 名称 | 时长 | 用途 |
|------|------|------|
| `--transition-fast` | 150ms | 微交互 |
| `--transition-normal` | 200ms | 标准过渡 |
| `--transition-slow` | 300ms | 大幅动画 |

### 缓动函数

```css
--ease-default: cubic-bezier(0.4, 0, 0.2, 1);
--ease-in: cubic-bezier(0.4, 0, 1, 1);
--ease-out: cubic-bezier(0, 0, 0.2, 1);
--ease-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55);
```

### 动画场景

1. **卡片悬浮**：上移4px + 阴影增强
2. **按钮点击**：轻微缩放(0.98)
3. **模态框出现**：淡入 + 从底部滑入
4. **分页切换**：淡入淡出
5. **加载状态**：旋转或脉冲

## 七、可访问性规范

### 色彩对比
- 主要文字：至少 4.5:1 对比度
- 次要文字：至少 3:1 对比度
- 大文本（18px+）：至少 3:1 对比度

### 交互反馈
- 所有可点击元素有明显的悬浮状态
- 焦点状态使用明显的轮廓线
- 加载状态有明确的视觉指示

### 键盘支持
- Tab 键导航所有交互元素
- Enter/Space 激活按钮
- Esc 关闭模态框
- 方向键浏览列表内容

## 八、实现注意事项

1. **CSS 变量化**：所有颜色使用 CSS 变量，便于主题切换
2. **模块化样式**：每个组件独立的 CSS Module
3. **组件复用**：通用组件抽取为独立组件
4. **动画性能**：使用 transform 和 opacity 避免重排
5. **图片优化**：封面图使用懒加载和缓存
6. **触摸优化**：移动端触摸区域至少 44x44px
